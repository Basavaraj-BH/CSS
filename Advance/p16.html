CSS Pseudo-elements..!

The CSS Pseudo-elements allow you to style the elements or parts of the elements without adding any IDs or classes to them.

CSS3 introdused a new double-colon (::) syntax for Pseudo-elements to distinguish between them and Pseudo-elements.

selector::Pseudo-elements{ property: value;}
